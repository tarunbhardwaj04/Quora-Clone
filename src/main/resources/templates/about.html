A robust, enterprise-grade REST API built using Spring Boot, designed to replicate core Q&A platform functionalities.

Architectural Overview

The project follows a rigorous Layered Architecture for separation of concerns:

  • Controller Layer: Handles HTTP requests/responses (REST).
  • Service Layer: Encapsulates business logic and validations.
  • Repository Layer: Manages data access via Spring Data JPA.
  • Entity Layer: Defines MySQL schema mappings.
  • DTO Layer: Secure data transfer objects for API contracts.

Technology Stack

  • Java 21 LTS
  • Spring Boot 3.5.x
  • MySQL 8.0
  • Flyway Migration

Key Features

  • Threaded Q&A: Deeply nested question and answer threads.
  • Database Integrity: Strict FK constraints and Flyway versioning.
  • Security: JWT-based Stateless Authentication
